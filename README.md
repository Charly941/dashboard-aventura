# Dashboard – Así comienza la aventura

## Cómo subir a GitHub Pages

1. Creá un repositorio nuevo en GitHub (puede ser privado o público)
2. Subí todos estos archivos al repo:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Andá a **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Guardá. En 1-2 minutos tenés la URL activa: `https://tuusuario.github.io/nombre-repo`

## Instalar como app
- **Celular (Android/Chrome)**: abrís la URL y te aparece "Agregar a pantalla de inicio"
- **Celular (iPhone/Safari)**: botón compartir → "Agregar a pantalla de inicio"
- **PC (Chrome/Edge)**: ícono de instalar en la barra de direcciones
